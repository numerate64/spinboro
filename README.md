# Spinboro

A static tribute website for the 5 AM Spinboro spin class community. The page combines karaoke-inspired visuals, cycling-studio energy, and interactive browser effects.

## Published Page

```text
https://numerate64.github.io/spinboro/
```

## Files

- `index.html` - page sections and content.
- `styles.css` - dark responsive design, neon accents, and animations.
- `script.js` - scroll animations, time-aware messages, rotating highlights, and interactive effects.

## Local Preview

```sh
python3 -m http.server 8000
```

This is a static tribute page with no backend, schedule system, or form submission workflow.
